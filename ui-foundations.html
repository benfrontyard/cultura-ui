Typography

Font stacks stay Cultura: Epilogue (--ch-font-display) for the largest display headings, General Sans (--ch-font-sans) for UI and smaller headings. Size and line-height primitives follow the Shopify Polaris font scale (11px–40px, 4px line-height rhythm), exposed as --ch-font-size-* and --ch-font-line-height-*. Marketing-style fluid steps remain: --ch-text-2xs–--ch-text-md; html root uses clamp() (~16–17.5px).

Sidebar nav states

Live states appear in the app shell. Each page sets data-ch-current on <body>; assets/js/app.js marks the matching data-ch-nav link with .ch-nav-active / Bootstrap .active and opens the parent <details class="ch-nav-group"> when the route lives inside a collapsible section. In the desktop sidebar and mobile drawer, the active row uses a neutral pill background (--ch-sidebar-nav-pill) with no left accent; other surfaces may still use the legacy blue rail (--ch-nav-active-indicator-width).
